1	ระบบลงทะเบียนสมาชิกและบริหารจัดการสัญญา
	1.1	หน้าฟอร์มบันทึกข้อมูลสมาชิกใหม่: ชื่อ, ที่อยู่, เบอร์โทร, อีเมล
	1.2	บันทึกข้อมูลผู้ติดต่อสำรองได้สูงสุด 3 รายต่อสมาชิก 1 ท่าน
	1.3	ระบบสร้างเลขที่สัญญาและบันทึกวันที่เริ่มสัญญา (อายุสัญญา 30 ปีอัตโนมัติ)
	1.4	การจัดเก็บไฟล์สัญญาดิจิทัล (Upload PDF/รูปภาพ) เข้าสู่ Google Drive

2	ระบบจัดการข้อมูลหลุม
	2.1	หน้าจอ Web App แสดงผังหลุมศพแบบตาราง (Grid View) ที่ปรับแต่ง Layout ได้
	2.2	แสดงสถานะหลุมด้วยสี (Color Coding): ว่าง (เขียว), บรรจุแล้ว (แดง), จองแล้ว แต่ยังไม่ได้บรรจุ (เหลือง), หมดสัญญา, ยังไม่ได้ย้าย (เทา), หลุม/ช่องยกเว้นค่าบำรุง (สีขาว)
	2.3	ระบบ Interactive: คลิกที่หลุมเพื่อดูรายละเอียดผู้ล่วงลับทั้ง 3 ชั้น และข้อมูลผู้ติดต่อได้ทันที

3	ระบบทะเบียนศพส่วนกลาง
	3.1	ฐานข้อมูลบันทึกรายชื่อผู้ล่วงลับ: ชื่อ-นามสกุล, วันที่นำเข้า, ตำแหน่งหลุม/ชั้น
	3.2	ระบบค้นหารายชื่อผู้ล่วงลับ (Search Engine) เพื่อระบุตำแหน่งหลุมได้รวดเร็ว
	3.3	ระบบหลุมรวม (Common Grave Archive): ทะเบียนสำหรับบันทึกศพที่ถูกย้ายออกจากหลุมเดิม (กรณีหมดสัญญา) เพื่อเก็บไว้เป็นประวัติการสืบค้น

4	ระบบจัดการการเงินและอนุมัติชำระเงิน
	4.1	ระบบบันทึกการชำระค่าบำรุงรายปี แยกตามปี
	4.2	ระบบกดอนุมัติ (Approve) เพื่อบันทึกเข้าฐานข้อมูล

5	ระบบบันทึกการเคลื่อนย้าย
	5.1	ฟังก์ชัน "ย้ายหลุม": ปรับเปลี่ยนสถานะหลุมเดิมให้ว่าง และบันทึกข้อมูลในหลุมใหม่
	5.2	Activity Log: บันทึกประวัติการย้ายศพ (ย้ายจากไหน ไปไหน วันที่เท่าไหร่ ใครเป็นคนทำรายการ)
	5.3	ฟังก์ชันย้ายเข้า "หลุมรวม" กรณีสิ้นสุดสัญญาและติดต่อญาติไม่ได้

6	ระบบบริหารพนักงานและบันทึกการทำงาน
	6.1	ระบบ Login แยกผู้ใช้งาน (เช่น พนักงาน 1, พนักงาน 2, เจ้าของ)
